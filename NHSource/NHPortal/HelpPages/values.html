OBD Malfunction Indicator Light Status

The value is composed of 2 separate digits, each one with its own set of values

The first digit is the OBD DLC Locate status whose values are:
0 = OK
1 = Missing
2 = Damaged
3 = Modified
4 = Tampered
5 = Obstructed
6 = Disconnected.

The second digit is the result of the OBD inspection:
0 = Mil Not Commanded On (Pass)
3 = Unable to communicate (Reject)
4 = Unready (Reject)
5 = MIL Commanded On (Reject)

A "6" in the second character means this vehicle is Ready Exempt i.e. the vehicle falls into a category of known vehicles that may have problems reporting readiness and so is exempt from readiness testing

A "7" in the second character means this vehicle is Comm Exempt i.e. the vehicle falls into a category of known vehicles with OBD communication problems and so is exempt from communications
