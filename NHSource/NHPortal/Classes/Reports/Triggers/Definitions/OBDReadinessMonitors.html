OBD II Readiness Monitors Definition

Readiness monitors displayed as 11 character string in the following order

Symbol Definition
1st character Misfire Monitor
2nd character Fuel System Monitor
3rd character Component Monitor
4th character Catalyst Monitor
5th character Heated Catalyst Monitor
6th character Evap System Monitor
7th character Air System Monitor
8th character A/C System Monitor
9th character Oxygen Sensor Monitor
10th character Heated Oxygen Sensor Monitor
11th character Misfire Monitor

With the following possible values

Symbol Definition
S Supported
U Unsupported

Example SSSSUSUUSSS indicates

Monitor Status
Misfire Monitor Supported
Fuel System Monitor Supported
Component Monitor Supported
Catalyst Monitor Supported
Heated Catalyst Monitor Unsupported
Evap System Monitor Supported
Air System Monitor Unsupported
A/C System Monitor Unsupported
Oxygen System Monitor Supported
Heated Oxygen Sensor Monitor Supported
EGR System Monitor Supported
